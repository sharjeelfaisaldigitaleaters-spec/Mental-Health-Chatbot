# Mental Health Support Chatbot

## Objective
Develop an AI-powered mental health support chatbot capable of generating empathetic and supportive conversational responses using transformer-based language models.

## Tools & Technologies Used
- Python
- Google Colab
- Hugging Face Transformers
- DistilGPT2
- EmpatheticDialogues Dataset
- PyTorch
- Matplotlib

## Features
- Transformer-based conversational AI
- Fine-tuned DistilGPT2 model
- Prompt engineering
- Text generation pipeline
- Emotionally supportive chatbot responses
- Training loss visualization
- Response length distribution analysis

## NLP Concepts Implemented
- Tokenization
- Language Model Fine-Tuning
- Text Generation
- Prompt Engineering
- Conversational AI
- Transformer Models

## Result
Successfully developed and tested a generative AI mental health chatbot capable of generating empathetic conversational responses using NLP and transformer-based language modeling techniques.
